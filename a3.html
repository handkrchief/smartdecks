SmartDecks API Documentation

User Management API

Runs on: index.js (Start with npm run dev)

Get All Users

Method: GET

Endpoint: /users

[
            { "userId": 1, "username": "testuser", "email": "test@gmail.com" }
        ]
Get User by ID

Method: GET

Endpoint: /users/:userId

{
            "userId": 1,
            "username": "testuser",
            "email": "testuser@gmail.com"
        }
Create a New User

Method: POST

Endpoint: /users

Content-Type: application/json

Request Body:

{ "username": "newuser", "email": "new@example.com" }

Success: 201 Created

{ "Success": "User created successfully.", "userId": 3 }

Deck Management API

List Decks for User

Method: GET

Endpoint: /users/:userId/decks

Create Deck for User

Method: POST

Endpoint: /users/:userId/decks

Request Body:

{ "name": "Sample Deck", "subject": "Math" }
Get Deck by ID

Method: GET

Endpoint: /decks/:deckId

Update Deck

Method: PUT

Endpoint: /decks/:deckId

Request Body:

{ "name": "Updated Deck", "subject": "Science" }
Delete Deck

Method: DELETE

Endpoint: /decks/:deckId

Flashcard Management API

List Cards in Deck

Method: GET

Endpoint: /decks/:deckId/cards

Create Card

Method: POST

Endpoint: /decks/:deckId/cards

Request Body:

{ "question": "What is 2+2?", "answer": "4" }
Update Card

Method: PUT

Endpoint: /cards/:cardId

Request Body:

{ "question": "Updated question", "answer": "Updated answer" }
Delete Card

Method: DELETE

Endpoint: /cards/:cardId
